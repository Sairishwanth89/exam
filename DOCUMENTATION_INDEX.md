## 📱 Mobile/Quick Reference

**One-page cheat sheet for common tasks:**

### Start Exam (Student)
```
1. Go to exam page
2. Click "Start Exam"
3. Face verification modal: Click "Start Capture"
4. Let system capture 3 faces (auto)
5. Exam begins
```

### Adjust Sensitivity
```python
# In ai/face_recognition_module.py
FACE_VERIFICATION_THRESHOLD = 0.55  # default
# 0.50 = stricter, 0.60 = more lenient
```

### Deploy
```bash
docker-compose down
docker-compose up -d --build
```

### Check Status
```bash
docker logs exam-ai-proctor
```

### View Report
```
http://localhost:3000/api/proctor/report/:username
```
